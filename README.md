# ZipBloat - ZIP文件体积放大工具

一个简单易用的ZIP文件体积放大工具，支持多种放大方式，包括ZIP炸弹模式。

## 功能特性

- 📁 **多种放大方式**：
  - 添加空块：在ZIP中添加大量空文件
  - 嵌套ZIP：创建多层嵌套的ZIP结构
  - 尾部追加垃圾数据：在ZIP文件尾部追加垃圾数据
  - ZIP炸弹模式：生成体积很小但解压后会爆炸的ZIP

- 🌍 **多语言支持**：中文和英文界面

- 📊 **实时进度显示**：带百分比的进度条

- ⚡ **操作防抖**：防止快速点击多次触发任务

- ❌ **支持取消**：生成过程中可以随时取消

- 📖 **内置使用说明**：详细的操作指南

## 系统要求

- Python 3.7+
- Windows 系统
- 足够的磁盘空间

## 安装方法

### 方法1：直接运行

1. 下载项目文件
2. 确保系统已安装Python 3.7+
3. 双击运行 `ZipBloat.py` 文件

### 方法2：命令行运行

```bash
# 克隆或下载项目后
cd ZipBloat
python ZipBloat.py
```

## 使用指南

1. **选择原ZIP文件**：点击「浏览」按钮选择需要放大的ZIP文件
2. **设置输出路径**：系统会自动生成输出路径，也可以手动修改
3. **设置目标体积**：输入需要放大到的目标体积（GB）
4. **选择放大方式**：从下拉菜单中选择合适的放大方式
5. **点击开始生成**：等待生成完成
6. **查看结果**：生成完成后会显示成功消息

## 放大方式说明

- **添加空块**：在ZIP文件中添加大量空文件，适用于常规测试
- **嵌套ZIP**：创建多层嵌套的ZIP结构，生成的文件体积较小
- **尾部追加垃圾数据**：在ZIP文件尾部追加垃圾数据，最稳定可靠
- **ZIP炸弹模式**：生成体积很小但解压后会爆炸的ZIP，用于测试解压防护

## 注意事项

- 目标体积建议设置在1-20GB之间，过大会耗时较长
- 确保输出路径有写入权限
- 使用ZIP炸弹模式时请谨慎，仅用于测试目的
- 生成过程中不要关闭程序窗口

## 常见问题

### Q: 任务栏图标显示为Python图标？
A: 这是Windows系统的正常现象，不影响使用

### Q: 权限拒绝错误？
A: 请确保输出路径有写入权限，建议选择桌面或文档文件夹

### Q: 磁盘空间不足？
A: 程序会自动检测磁盘空间，不足时会提示是否继续
